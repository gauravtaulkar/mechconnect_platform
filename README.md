# MechConnect Platform

## Overview
Full-stack web application built using:
- Spring Boot (Backend)
- React (Frontend)
- MySQL (Database)

## Project Structure
backend/   → Spring Boot API  
frontend/  → React UI  
## How to Run

### Backend
cd backend  
./mvnw spring-boot:run  

### Frontend
cd frontend  
npm install  
npm start  

## Features
- REST API integration
- Full-stack architecture
